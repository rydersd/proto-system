Analytics Dashboard
Summary
A dashboard with two functional SVG charts (bar, donut) and a data table that regenerate random data on each page reload. Demonstrates pure SVG chart rendering and table components using framework design tokens — no external libraries. KPI cards show randomized metrics with trend indicators.
JTBD
Product manager needs a quick overview of key business metrics — revenue trends, user growth, traffic sources, and conversion funnel health — all in one glanceable view.
Design Spec
Single-column layout at 1200px max-width. 4-up KPI grid at top. Row of 2-column chart cards (bar + donut), then a full-width data table below. Charts use responsive SVG with viewBox. Table uses wf-table with status badges and color-coded growth values. Colors pulled from CSS custom properties at runtime. Paper utilities (tape on bar chart, pin on donut, sketch on table) add wireframe texture.

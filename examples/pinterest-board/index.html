Summary
A Pinterest-style masonry mood board for collecting and organizing design inspiration. Uses CSS multi-column layout to create the staggered card grid. Each pin is a transparent card container where individual elements (image, title, badge, avatar, heart) appear as separate paper scraps with their own shadows and slight rotations — creating a corkboard aesthetic.
JTBD
Designer needs a visual space to collect, categorize, and browse design references — quickly scanning inspiration by category (UI, typography, color, layout, motion) and saving favorites for later use in projects.
Design Spec
Top bar with title, search input, and primary add-pin button. Filter pill row with category toggles (one active shown with accent background). 4-column CSS masonry grid (responsive: 3 cols at 1024px, 2 cols at 768px) using column-count and break-inside:avoid. 12 pin cards with varied placeholder heights (120-300px) for visual rhythm. Each card is a transparent container (.pin-card) with individual paper scrap children (.pin-scrap): image scrap with paper shadow and optional tape/pin/sketch utilities, title strip on its own paper, badge as tiny paper tag, and avatar+heart as small action scraps. Paper utilities distributed for texture: wf-pin (pins 1, 4), wf-tape (pins 2, 11), wf-sketch (pins 5, 12), wf-stacked (pin 7), wf-torn-bottom (pin 8). Confidence attributes mark design certainty. Load More button centered at bottom.

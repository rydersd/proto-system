Minimal Dashboard Layout

A clean, grid-based dashboard layout that prioritizes whitespace and content hierarchy. Uses a 12-column grid with KPI cards at the top, a primary chart area, and a secondary data table below. The sidebar is collapsible and houses navigation and filter controls.

Summary
A data-rich table view for managing customer records. Includes search, multi-filter toolbar, bulk actions, KPI summary cards, tabbed filtering by status, and paginated results. Designed to showcase the proto-framework's table, badge, form, tab, and card components working together in a realistic internal portal layout.
JTBD
Customer success manager needs to quickly find, filter, and act on customer records — identifying at-risk accounts, exporting data for reports, and managing individual customer details.
Design Spec
Single-column layout at 1200px max-width. Toolbar card with search + filters + bulk actions. 4-up KPI grid. Tab bar for status filtering. Full-width data table with checkbox selection, status badges, color-coded health scores, and inline action buttons. Pagination footer with page buttons. Paper utilities (tape on toolbar, pin on table card, sketch on MRR KPI) add wireframe texture. Confidence attributes mark design certainty — churned-row layout and MRR KPI are flagged as uncertain/partial for stakeholder review.

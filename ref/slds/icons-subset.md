# SLDS Icon Catalog

> Salesforce Lightning Design System SVG icons for wireframes. Source: `@salesforce-ux/design-system@2.25.4`.

## SLDS Naming Convention

Icons follow the pattern `{category}:{name}`:
- **utility** — Small UI icons (16px default). Used in buttons, badges, form controls.
- **standard** — Object icons (larger, with colored backgrounds in SLDS). Used in record headers.

In Lightning: `<lightning-icon icon-name="utility:check">`. In wireframes: inline SVGs with `.wf-icon`.

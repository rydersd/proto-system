What is a Job to Be Done?

A JTBD describes what a person is trying to accomplish in a given context, independent of the solution. Format: "When [situation], I want to [motivation] so I can [outcome]." Each job maps to one or more wireframe pages that address it.

Summary

The JTBD Hub consolidates all persona-specific job statements from individual wireframe design notes into one browsable reference. Each job links back to the wireframe page(s) that address it.

Design Specification

Primary Content

Persona blocks with numbered JTBD items. Each item includes the job statement and links to relevant screens.

Interactive Elements

  • View-control nav — links to sibling reference pages
  • Screen links — navigate to wireframe pages that address each job
  • Tab system — segment by audience (if multi-audience)

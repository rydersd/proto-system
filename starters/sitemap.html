Summary

The Sitemap page provides a visual information architecture overview of the entire prototype. It serves as the landing page and navigation hub, showing all pages organized by section with their key components.

Design Specification

Primary Content

Branch containers grouped by SECTIONS from project-data.js. Each branch contains page nodes with type icons, labels (linked to pages), descriptions, and expandable component inventories.
